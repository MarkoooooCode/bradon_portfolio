USER RESEARCH: SUMMARY

I conducted interviews and created empathy maps to understand the users I’m designing for, as well as their needs. The primary user group was for 12 and older who’s looking for ingredients and procedure in a certain dish.



This user group confirmed my initial assumptions about getting unwanted recommendations when searching for something sweet. During the research, it revealed that some users did not care for the other recommendations.

USER RESEARCH: PAIN POINTS

1

UNWANTED SEARCHES

User wanted searches that pertained to what it is they were searching for

2

ONLY BIG COMPANY’S

It only showed big companies. Multiple users wanted the option to see private and smaller businesses.
